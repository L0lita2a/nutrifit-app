## Professional Summary

**NutriFit** is a comprehensive and intuitive mobile application designed to streamline personal health and wellness tracking. The application empowers users to accurately monitor their daily habits, analyze their progress, and maintain a sustainable lifestyle. By prioritizing a seamless user experience, responsive design, and robust data synchronization, NutriFit provides an efficient tool for achieving long-term health and fitness objectives.

## Key Features

* **Firebase Auth**: Secure, reliable user authentication and personalized session management.
* **Firestore**: Real-time cloud database integration ensuring instantaneous synchronization and persistent data storage across devices.
* **Multi-currency**: Dynamic currency support enabling users to manage and track expenditures in various global currencies effortlessly.
* **fl_chart**: Interactive, high-performance visual data representation, utilizing charts for an in-depth statistical analysis of user data.

## Technical Architecture

* **Provider**: Implemented for efficient, scalable state management. It allows for reactive UI updates and clean separation of business logic from the presentation layer, ensuring a highly maintainable codebase.
* **Firebase**: Serves as the robust backend infrastructure, handling secure user authentication processes and providing a scalable NoSQL cloud database for real-time data persistence.

## How to Install & Run

Ensure that you have the [Flutter SDK](https://docs.flutter.dev/get-started/install) installed on your machine.

1. **Clone the repository**
   ```bash
   git clone <repository_url>
   ```
2. **Navigate to the project directory**
   ```bash
   cd nutrifit
   ```
3. **Install dependencies**
   ```bash
   flutter pub get
   ```
4. **Run the application**
   ```bash
   flutter run
   ```
